# MediaMetrics

Pop culture trend analytics dashboard that analyzes search interest data for artists, movies, and TV shows.

## Tech Stack
- React
- Flask
- SQLite
- Python
- Google Trends (pytrends)

## MVP Features
- Search media topics (artists, movies, TV shows)
- Visualize interest over time
- Classify trends as rising, stable, or declining
- Save trends for later analysis
